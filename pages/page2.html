Para que se utiliza un cable cruzado en la red

Conectar Dos Computadoras: Se utiliza para conectar directamente dos computadoras sin necesidad de un switch o hub. El cable cruzado permite que los dispositivos envíen y reciban señales de manera correcta al cruzar las líneas de transmisión y recepción.

Conectar Dispositivos de Red: Puede conectar directamente dos switches, routers o hubs entre sí para establecer una red sin un dispositivo adicional que actúe como intermediario.

Transferencia de Archivos: Facilita la transferencia de archivos entre dispositivos conectados directamente, como en un entorno de red local donde no se dispone de un switch o router.
